`growkar` is a Bioconductor-oriented software package for high-throughput
microbial phenotyping from plate-based growth assays. It uses
`SummarizedExperiment` and related Bioconductor infrastructure to represent OD
measurements, sample annotations, and derived growth phenotypes such as growth
rate, doubling time, exponential-phase windows, and fitted model summaries.
This allows growth-based phenotyping data to be integrated directly with
transcriptomic, genomic, and other omics datasets in downstream functional
genomics and microbial systems biology workflows. The package addresses a gap
in the current Bioconductor ecosystem by providing dedicated infrastructure and
analysis methods for microbial growth assays rather than treating these
phenotypes as ad hoc external summaries.
